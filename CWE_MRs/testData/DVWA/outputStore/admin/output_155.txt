Damn Vulnerable Web App (DVWA) is a PHP/MySQL web application that is damn vulnerable. WARNING! Damn Vulnerable Web App is damn vulnerable! Do not upload it to your hosting provider's public html folder or any internet facing web server as it will be compromised.
